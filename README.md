# EduPath India

## Live site - [paths-after-10th.pages.dev](https://paths-after-10th.pages.dev)

Interactive roadmap for Indian education and career paths.

## Features

- Visual graph-based education roadmap (Class 10 → PhD)
- Programming career paths with language selection helper
- Search functionality to find careers and courses
- Mobile-friendly list view and desktop map view

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.
